```

## Opdracht 2

* Maak een variabel `input` aan.
* Controleer of de `input` een positief getal is en tel af naar 0 met behulp van een While-loop.
* Toon in de console de getallen van het aftellen.

### Verwachte output:
Als input -1 is:
```
Voer een positief getal in:

```
Voer een positief getal in:
5
4
3
2
1
0

// Vraag een positief getal aan de gebruiker
Console.WriteLine("Voer een positief getal in:");
int getal = int.Parse(Console.ReadLine);

// Controleer of het getal positief is
while (...){

}

// Aftellen naar 0
while (...){

}
```

> [!info]- Mogelijke uitwerking
> ``` csharp

## Opdracht 3
Zet de volgende casus om naar een While loop.

### Specificaties
* Vraag de gebruiker om een reeks positieve getallen in te voeren.
* Tel de ingevoerde getallen bij elkaar op.
* Het programma stopt zodra de gebruiker "0" invoert.
* Toon de som van alle ingevoerde getallen.

### Verwachte output:
Als invoer bijvoorbeeld `5, 3, 2, 0` is:
```
Voer een positief getal in (0 om te stoppen):  
5  
Voer een positief getal in (0 om te stoppen):  
3  
Voer een positief getal in (0 om te stoppen):  
2  
Voer een positief getal in (0 om te stoppen):  
0  
Het totaal van alle ingevoerde getallen is: 10  
```

### Nu jij:
``` csharp runner
using System;

// Variabel om het totaal bij te houden
int totaal = 0;

// Vraag een positief getal aan de gebruiker
Console.WriteLine("Voer een positief getal in (0 om te stoppen): ");
int getal = int.Parse(Console.ReadLine());

// While-lus om getallen te blijven vragen
while (...){
	// Schrijf je code hier
}

// Print het totaal
```

> [!info]- Mogelijke uitwerking
> ``` csharp
> using System;
>
>// Variabel om het totaal bij te houden
>int totaal = 0;
>
>// Vraag een positief getal aan de gebruiker
>Console.WriteLine("Voer een positief getal in (0 om te stoppen): ");
>int getal = int.Parse(Console.ReadLine());
>
>// While-lus om getallen te blijven vragen
>while (getal != 0)
>{
>    totaal += getal;  // Voeg het ingevoerde getal toe aan het totaal
>    Console.WriteLine("Voer een positief getal in (0 om te stoppen): ");
>    getal = int.Parse(Console.ReadLine());  // Vraag opnieuw om een getal
>}
>
>// Print het totaal
>Console.WriteLine("Het totaal van alle ingevoerde getallen is: " + totaal);
> ```

## Opdracht 5
Schrijf een programma dat de gebruiker vraagt om een getal in te voeren. Het programma vermenigvuldigt dat getal steeds met 2 totdat het resultaat groter is dan 100. Het programma moet de gebruiker ook laten zien hoeveel keer het getal is vermenigvuldigd voordat het resultaat groter is dan 100.

### Specificaties
* Vraag de gebruiker om een getal in te voeren
* Vermenigvuldig het getal met `2` tot het resultaat groter is dan `100`
* Toon hoe vaak het getal is vermenigvuldigd

### Verwachte output:
Als de invoer 5 is:
```
Het getal is 7 keer vermenigvuldigd totdat het groter was dan 100.
```
Als de invoer 10 is:
```
Het getal is 6 keer vermenigvuldigd totdat het groter was dan 100.
```

### Nu jij:
``` csharp runner
using System;

int number;
int count = 0;

Console.WriteLine("Voer een getal in:);
number = int.Parse(Console.ReadLine());

do{

} while();
```

> [!info]- Mogelijke uitwerking
> ``` csharp
> using System;
>
>int number;
>int count = 0;
>
>Console.WriteLine("Voer een getal in: ");
>number = int.Parse(Console.ReadLine());
>
>do
>{
>	number *= 2;
>	count++;
>} while (number <= 100);
>
>Console.WriteLine($"Het getal is {count} keer vermenigvuldigd totdat het groter was dan 100.");
>```
